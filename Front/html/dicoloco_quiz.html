Mini Quiz



Bienvenue sur la page de notre mini quiz


Les règles sont simples :

  • Vous devez pour chaque question, choisir le synonyme correspondant au mot affiché
  • Une seule réponse possible par question
  • Limite de temps : aucune

     Vous pouvez commencer dès maintenant en appuyant sur le bouton "Commencer" ou rejouer le quiz avec le bouton "Recommencer".


     Voili voulou, Bonne chance c'est à vous de jouer :)
